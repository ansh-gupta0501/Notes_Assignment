# AWS ENVIRONMENTAL VARIABLES

- set env variables in configuration > environmental variables .
- you will find those environmental variables in your execution environment 

```javascript
export const handler = (event,context) =>{
    const name = 'ansh'
    const response = `my name is ${name}`
    console.log(process.env)
    return {
        'statusCode' : 200,
        'body' : JSON.stringify(response)
    }
}
```

- now updates the code 
```javascript
export const handler = (event,context) =>{
    const name = process.env.myname
    const age = process.env.age

    const response = `my name is ${name} and my age is ${age}`

   
    return {
        'statusCode' : 200,
        'body' : JSON.stringify(response)
    }
}
```
